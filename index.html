SpaceHeyMods

This is an open-source extension with publicly available code that improves SpaceHey and fixes bugs.

Download Extension

Features

  • Send messages with Enter in SpaceHey IM

Installation

  1. Download and extract the ZIP.
  2. Open chrome://extensions
  3. Enable Developer Mode
  4. Click Load unpacked
  5. Select the extension folder
