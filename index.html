SpaceHeyMods

This is an open-source extension with publicly available code that improves SpaceHey and fixes bugs.

Download Extension

Features

  • Send messages with Enter in SpaceHey IM
  • Friend Request Notifications - Get notified when you receive a new friend request
  • Message Notifications - Get notified when someone sends you a message

How Notifications Work

The extension checks for updates in the background every 1 minute. This helps you stay updated without needing to refresh the page constantly.

  • Background Polling: The extension briefly visits your Home page to see if your message or friend request count has increased.
  • Direct Detection: If you are already browsing SpaceHey, the extension detects new notifications instantly.
  • Minimal Delay: Notifications appear within 0-60 seconds of a new event.

Installation

  1. Download and extract the ZIP.
  2. Open chrome://extensions
  3. Enable Developer Mode
  4. Click Load unpacked
  5. Select the extension folder
